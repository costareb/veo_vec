How to install and run this app:

1) Download and install Docker Desktop:
   https://www.docker.com/products/docker-desktop/

2) Unzip this folder 'veovec_proto' somewhere

3) Put the folder "2025-07-25_FAOLEX_Kolumbien_gesamt" inside the empty folder called 'pdf_data' which is inside veovec_proto.

4) Start the app:
   - Open Docker Desktop and wait until it says “engine running” (on the bottom left of the docker-window).
   - Double click on the file “setup_start.bat”, a file inside veovec_proto.
   - This will set up the application, which might take some time as all the models and program packages have to be downloaded; 
   - In future, you can double click the other file "quick_start.bat", which then should only take a few seconds. 

5) In case it does not open automatically, open your browser at:
   http://localhost:8501/

6) You should see the welcome screen now, left to which you find 6 tabs. The rightmost tab 'help' has a button 'readme' with further instructions about the app itself.
